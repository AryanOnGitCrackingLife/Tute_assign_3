# Flask + MongoDB App

A simple web application that demonstrates how to integrate Flask with MongoDB Atlas for data insertion and retrieval.

## Features

- Web form to submit data to MongoDB
- REST API endpoint to view stored data
- MongoDB Atlas cloud database integration

## Tech Stack

- **Backend**: Flask (Python)
- **Database**: MongoDB Atlas
- **Frontend**: HTML forms
